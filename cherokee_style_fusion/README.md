# Cherokee Style Fusion Eco Font Lab

This folder is a self-contained experiment for Cherokee eco-font generation by:

1. fusing styles among existing Cherokee Unicode fonts,
2. adding a cross-style aesthetic prior,
3. ranking candidates by ink saving, readability margin, topology, and aesthetic balance.

It intentionally does not modify any other project folder.

## Method

The pipeline treats Cherokee glyph structure as the content anchor. Other Cherokee fonts provide style references for the same Unicode codepoint. Each candidate is generated by:

- rendering a base Cherokee glyph,
- rendering a same-character style glyph from another Cherokee font,
- aligning the style glyph to the base glyph bounding box,
- blending and morphologically adjusting stroke weight, width, and slant,
- optionally adding eco cuts such as inline channels or stencil cuts,
- scoring the result with an aesthetic/readability/ink objective.

The key design decision is that language genealogy is not used as the main signal. Font generation depends more on glyph geometry than linguistic ancestry, so the content structure stays Cherokee while external style ideas are represented only as aesthetic priors.

## Folder Layout

- `configs/default.json`: default experiment configuration.
- `charsets/cherokee_demo.txt`: default Cherokee sample characters.
- `scripts/download_fonts.py`: downloads a small public-font candidate bank into `data/fonts`.
- `scripts/run_inventory.py`: checks Cherokee Unicode coverage for fonts in `data/fonts`.
- `scripts/generate_candidates.py`: generates style-fusion candidates and contact sheets.
- `src/cherokee_style_fusion`: implementation modules.
- `outputs`: generated samples.

## Quick Start

```bash
cd /home/woojye2020/decs_jupyter_lab/eco-font/cherokee_style_fusion
python scripts/download_fonts.py --preset demo
python scripts/run_inventory.py --fonts-dir data/fonts --out outputs/font_inventory.json
python scripts/generate_candidates.py --config configs/default.json
```

The main output is:

```text
outputs/demo/contact_sheet.png
outputs/demo/sheets/*.png
outputs/demo/candidates/*/*.png
outputs/demo/manifest.jsonl
outputs/demo/summary.json
```

## Notes

- `Noto Sans Cherokee` is the safest default baseline because it is available under the SIL Open Font License through Google Fonts.
- Some public font indexes mix open, free, legacy, and system fonts. Use `font_sources.json` as an experiment source list, not as redistribution/legal approval.
- The current readability metric is an ensemble template-recognition margin over Cherokee fonts. It is not a replacement for a production Cherokee OCR model.
